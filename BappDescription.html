Click-Jackalope generates ready-to-use HTML iframe proof-of-concept (POC) pages for testing clickjacking vulnerabilities. It enables testers to quickly create and open POCs directly from within Burp Suite without leaving the workflow.

Features

  • Create HTML iframe clickjacking POCs directly from Burp Suite
  • Adds a "Click-Jackalope" tab for manual URL entry and POC generation
  • Right-click context menu option: "Create Click-Jackalope POC" from selected requests
  • Auto-generates filenames based on target host for organized evidence
  • Optionally opens the generated file in your default system browser
  • Escapes embedded URLs to prevent HTML injection and operates fully offline
  • Lightweight and dependency-free (pure Jython implementation)

Usage

  1. Navigate to the Click-Jackalope tab
  2. Enter or paste the target URL, or right-click a request and choose "Create Click-Jackalope POC"
  3. Optionally edit the output filename
  4. Click Generate to create a local HTML file, or Generate & Open to open it immediately in your browser
  5. Use the generated page to verify whether the target allows framing and to demonstrate missing X-Frame-Options or weak Content-Security-Policy headers
